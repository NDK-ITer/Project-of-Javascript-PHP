# Hệ thống tuyển dụng và tìm việc "Whats job"

Hệ thống được thiết kế theo mô hình "Client To Server".
Chức năng bao gồm:
   Đăng nhập và đăng ký tài khoản.
   Cập nhập thông tin CV và bằng cấp.
   Tạo tin tuyển dụng, xét duyệt tuyển dụng.
   Nộp CV và ứng tuyển.
